SpyMe TensorFlow.js

About

This application uses a webcam to detect objects, specifically focusing on identifying people through a machine learning model. When a person is detected in the video feed, it sends an email notification.

The COCO-SSD (Common Objects in Context - Single Shot MultiBox Detector) model is a deep learning model used for object detection. It's trained on the Microsoft Common Objects in Context (COCO) dataset, which contains a wide variety of objects commonly found in everyday scenes. The model is capable of detecting and classifying a range of objects, making it a versatile tool for various computer vision applications, including object recognition and image analysis.

HOW TO USE

Please, wait for the model to load before clicking the button to enable the webcam.

Hold some objects up close to your webcam to get a real-time classification. When ready click "enable webcam" and accept access to the webcam when the browser asks (check the top left of your window)
